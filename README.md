## Meta Ray-Ban Display app

A HUD-style version of the portfolio built for Meta Ray-Ban Display
glasses lives at [`/display-app`](https://sebastianboscan.com/display-app).
It uses a fixed 600x600 viewport with arrow-key + Enter navigation
(Neural Band gestures map to keyboard events). The main site shows a
dismissible toast pointing visitors to it.
